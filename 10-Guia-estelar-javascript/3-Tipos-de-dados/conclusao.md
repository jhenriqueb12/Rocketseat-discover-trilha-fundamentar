# Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados:

* __Data types:__
  * Primitive / Primitive value
  * Structural
  * Structural primiteve

## Primitivos

* String
* Number
* Boolean
* Undefined
* Symbol
* BigInt

## Estruturais

* Object
  * Array
  * Map
  * Set
  * Date
  ...
* Function

## Primitivo Estrutural / Structural Root Primitive
* null
